  |  Features (see Explanations for details):
  |
  |  - 1 Bitmap, 1 Screen-RAM, Color-RAM
  |  - Multicolor pixels
  |
  |  Memory-Structure ($4000-$6800, 10240 Bytes, 41 Blocks):
  |
  |   $4000 - $5F3F: Bitmap
  |   $6000 - $63E7: Screen-RAM
  |   $6400 - $67E7: Color-RAM
  |   $67FF        : Background-Color
