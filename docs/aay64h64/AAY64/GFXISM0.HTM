+------------------------------------------------------------------------
  |
  |  Image System
  |
  +------------------------------------------------------------------------
  |
  |  Features (see Explanations for details):
  |
  |  - 1 Bitmap, 1 Screen-RAM, Color-RAM
  |  - Multicolor pixels
  |
  |  Memory-Structure ($3C00-$63E8, 10216 Bytes, 41 Blocks):
  |
  |   $3C00 - $3FE7: Color-RAM
  |   $4000 - $5F3F: Bitmap
  |   $5FFF        : Background-Color
  |   $6000 - $63E7: Screen-RAM
  |
  +------------------------------------------------------------------------
