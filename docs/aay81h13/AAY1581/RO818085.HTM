Sucht Drivenummer in der Befehlszeile ($c1d1)
Die Position wird in $0291 gespeichert (0: keine Drivenummer)
E: y: Position, an der die Suche beginnen soll
Die Routine schaltet die Drive-LED an

Einsprung von $8EC5, $96BB, $B262:

8085: A2 00     LDX #$00         Flag fuer 'keine Drivenummer'
8087: 8E 91 02  STX $0291
808A: A9 3A     LDA #$3A         Kommandozeile nach ':' durchsuchen
808C: 20 1C 81  JSR $811C
808F: F0 05     BEQ $8096        nicht gefunden, dann ==>
8091: 88        DEY
8092: 88        DEY              Position der Drivenummer merken
8093: 8C 91 02  STY $0291        (steht genau vor dem ':')

Einsprung von $808F:

8096: 4C 51 82  JMP $8251        LED einschalten
